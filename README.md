# UXV
## Ubuntu18.04 Xfce4 Vncserver

### Manual Installation
```
git clone https://github.com/sadikkuzu/UXV.git
cd UXV
./vnc-xfce4.sh
```

### Direct Installation
```bash
$ bash -c "$(wget -qO- https://raw.githubusercontent.com/sadikkuzu/UXV/master/install)"
```
